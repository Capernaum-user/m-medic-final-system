# MDTS 통합 진단 앱 사용 안내

> 파일: `MDTS_통합진단앱.py` / 런처: `MDTS_실행.bat`
> 위치: `04_통합시스템/`

이미지 1장을 업로드하면 **딥러닝 / 머신러닝 / 종합 모델 3가지**를 동시에 호출해 결과를 출력하는 단일 실행 프로그램입니다.

---

## 1. 빠른 실행 (Windows)

1. `04_통합시스템` 폴더로 이동
2. **`MDTS_실행.bat`** 더블클릭
3. 패키지 자동 확인 → 서버 기동 → 브라우저 자동 오픈
4. 브라우저에 `http://127.0.0.1:8009` 페이지가 열림

> 첫 실행 시 `torch` 등 패키지가 없으면 자동 설치를 시도합니다. 인터넷이 필요합니다.

### CLI 직접 실행

```bash
cd 04_통합시스템
python MDTS_통합진단앱.py
# 옵션:
python MDTS_통합진단앱.py --port 8010 --no-browser
```

종료는 콘솔 창에서 `Ctrl+C`.

## 2. 화면 구성

| 영역 | 내용 |
|------|------|
| ① 외상 이미지 업로드 | 드래그/클릭으로 이미지 선택 (JPG/PNG/BMP/WEBP, 최대 10MB) |
| ② 사고 정보 입력 | 연도, 사고유형, 선박종류, 사고원인, 발생해역 (학습 CSV에서 자동 채움) |
| ③ 분석 결과 | DL / ML / 종합 3개 카드 + 진단 요약 + JSON 리포트 경로 |

---

## 3. 사용하는 모델 3종

| 구분 | 모델 | 입력 | 출력 |
|------|------|------|------|
| ① 딥러닝 | MobileNetV3 Large (`mobilenet_v3_wound_best.pth`) | 외상 이미지 | 외상 6종 분류 + 신뢰도/Top-3/유사도 |
| ② 머신러닝 | RandomForest (`model_randomforest_binary.pkl`) | 사고 정보 5개 | 사상자 발생 확률/위험도 |
| ③ 종합 | DL + ML 가중 결합 (DL 60% + ML 40%) | 위 두 결과 | 최종 위험등급 + 응급 조치 + 법적 근거 |

### 외상 6종 ↔ 위험도

| 클래스 | 표시명 | 위험도 |
|--------|--------|--------|
| Abrasions | 찰과상 | LOW |
| Bruises | 타박상 | LOW |
| Cut | 절상 | MEDIUM |
| Burns | 화상 | HIGH |
| Laceration | 열상 | HIGH |
| Stab_wound | 자창 | CRITICAL |

### 종합 점수 → 등급

`final_score = (DL_위험도/4) × 0.6 + ML_사상자확률 × 0.4`, 그 다음 4점 만점으로 환산.

| 점수 구간 | 종합 위험등급 |
|-----------|---------------|
| ≥ 3.5 | CRITICAL |
| ≥ 2.5 | HIGH |
| ≥ 1.5 | MEDIUM |
| < 1.5 | LOW |

## 4. 확인불가(OOD) 판정 기준

다음 중 **하나라도 해당되면** 종합 진단을 보류하고 "확인불가 이미지"로 표시합니다.

| 기준 | 임계값 |
|------|--------|
| DL 모델 신뢰도 | < 70% |
| 1위와 2위 후보 차이 | < 20% |
| 학습 데이터 feature 유사도 | < 50% (`wound_ood_reference.json` 기준) |

확인불가 시 안내 메시지: *"학습된 6종 상처 이미지와 충분히 유사하지 않습니다. 상처 부위를 중심으로 더 밝고 선명하게 재촬영하거나 전문가 확인이 필요합니다."*

## 5. 진단 리포트 자동 저장

매 분석마다 다음 위치에 JSON 리포트가 저장됩니다.

```
04_통합시스템/진단결과_샘플JSON/report_YYYYMMDD_HHMMSS.json
```

리포트 안에는 DL/ML/종합 결과, 입력값, 신뢰도, OOD 정보, 응급 조치 지침이 포함됩니다.

## 7. 자주 쓰는 트러블슈팅

| 증상 | 해결 |
|------|------|
| 브라우저가 열리지 않음 | 콘솔 창에 표시된 URL을 직접 입력. 또는 `--no-browser` 옵션 후 수동 접속 |
| `포트 사용 중` 에러 | `python MDTS_통합진단앱.py --port 8010` 처럼 다른 포트로 실행 |
| ML 카테고리가 비어 있음 | `marine_accidents_augmented_*.csv` 경로 확인 (이름 변경 시 코드 상수 수정 필요) |
| GPU에서 메모리 부족 | CPU에서도 동작합니다. 환경변수 `CUDA_VISIBLE_DEVICES=` 로 강제 CPU 사용 |
