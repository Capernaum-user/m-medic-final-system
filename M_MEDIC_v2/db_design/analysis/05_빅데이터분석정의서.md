# [M-MEDIC v2] 빅데이터 분석 정의서 (Big Data Analysis Definition)

**작성일:** 2026년 4월 22일  
**버전:** v2.1 (데이터 보강 완료)

## 1. 분석 개요
본 프로젝트는 해상 사고 현장에서 발생하는 외상(상처) 이미지를 AI로 식별하여 적절한 응급처치 가이드를 제공하는 것을 목적으로 함.

## 2. 데이터 수집 현황 (Wound Image Dataset)
기존 데이터셋의 불균형을 해소하고 모델의 정확도를 높이기 위해 외부 공공 데이터(Kaggle) 수집 및 데이터 증강(Augmentation)을 통해 6개 핵심 클래스를 보강함.

### 2.1. 최종 데이터 구성
- **총 데이터 수:** 4,212장 (기존 1,162장에서 약 3.6배 증가)
- **데이터 위치:** `M_MEDIC_v2/01_data/raw/wound_classification`

| 분석 대상 (클래스) | 국문 명칭 | 이미지 개수 | 증강 비율 | 비고 |
| :--- | :--- | :---: | :---: | :--- |
| **Abrasions** | 찰과상 | 668 | 2.7배 | 수집+증강 |
| **Bruises** | 타박상 | 972 | 2.7배 | 수집+증강 |
| **Burns** | 화상 | 504 | 2.6배 | 수집+증강 |
| **Cut** | 절창 | 600 | 4.0배 | 수집+증강(2x) |
| **Laceration** | 열창 | 732 | 4.0배 | 수집+증강(2x) |
| **Stab_wound** | 자창 | 736 | 32.0배 | 수집+집중증강(15x) |
| **Total** | **합계** | **4,212** | - | - |

## 3. 데이터 전처리 및 정제 (Data Cleaning)
- **의미 없는 데이터 정리:** 프로젝트 목적에 맞지 않는 당뇨병성 상처, 욕창, 내성발톱 등 10여 종의 데이터는 `deprecated/trash` 폴더로 이동하여 분석 대상에서 제외함.
- **데이터 증강 기법:** 
  - 부족한 클래스(특히 자창)에 대해 랜덤 회전(0~360도), 좌우 반전, 밝기 조절(0.7~1.3배) 등을 적용하여 학습 데이터의 다양성 확보.

## 4. 분석 모델링 전략
- **알고리즘:** MobileNet V3 (Lightweight Deep Learning Model)
- **입력 데이터:** 224x224 RGB Image
- **분석 목표:** 6개 상처 유형에 대해 Top-1 Accuracy 90% 이상 달성

## 5. 특이사항
- 자창(Stab_wound) 데이터의 원본 부족 현상을 집중 증강을 통해 해결하였으며, 추후 실제 해상 사고 사례 데이터를 추가 확보하여 모델 성능을 지속적으로 고도화할 예정임.
